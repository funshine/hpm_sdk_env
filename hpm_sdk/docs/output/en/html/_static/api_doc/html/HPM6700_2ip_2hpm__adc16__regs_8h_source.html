179 /*
180  * VALID (RO)
181  *
182  * set after conversion finished if wait_dis is set, cleared after software read.
183  * The first time read with 0 will trigger one new conversion.
184  * If SW read other channel when one channel conversion is in progress, it will not trigger new conversion at other channel, and will get old result with valid 0, also with read_cflct interrupt status bit set.
